

imageLocations.mat : First column is images. Second column is the corresponding locations. 

imageTagsFile.mat : First column is images. Second column is the corresponding tags. 

imageSimMat.mat : Similarity matrix of the images (500x500). 

locatioinSimMat.mat : Similarity matrix of the locations (99x99).

tagSimMat.mat: Similarity matrix of the tags (590x590).

dump.imgs : Information of the images. 

dump.tags : First column is image. Second column is corresponding tags of the image. 

IDvsImage.map : First column is ID of images. Second column is the corresponding images. 

IDvsLocation.map : First column is ID of locations. Second and third columns are the coordinates of the locations. 

IDvsTag.map : First column is ID of tags. Second column is the name of the tags.
